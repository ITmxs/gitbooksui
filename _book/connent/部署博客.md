## **四、部署个人博客（建站）**

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

```
hexo init jianguo
 cd jianguo
 npm install
```

新建完成后，指定文件夹的目录如下：

```
.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes
```

在这儿，不讲解过多的hexo命令以及它的配置

直接启动服务器。默认情况下，访问网址为： `http://localhost:4000/`。

```
hexo server
```
